# go-sr
Software-based subset of Opengl 1 in 600 lines of Go

## Features
- Quad drawing (points, lines, filled)
- Per-face lighting (directional and point)

## Usage 
```
git clone https://github.com/hortencio-main/go-sr
cd go-sr
go run examples/demo2.go
```
